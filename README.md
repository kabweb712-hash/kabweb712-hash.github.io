## Stack technique

- **HTML5 / CSS3** — sémantique propre, accessibilité (skip link, attributs ARIA)
- **JavaScript vanilla** — pas de framework, pas de build, performance maximale
- **CSS Grid + Flexbox** — layout bento responsive (12 colonnes desktop, 6 sur tablette, 1 sur mobile)
- **Hébergement** : GitHub Pages (déploiement continu via push sur `main`)
- **Typographies** : Instrument Serif + DM Sans (Google Fonts)
- **Thème** : dark mode + light mode avec toggle

## Structure du projet

```
kabweb712-hash.github.io/
```
